virtual bool UpdateCameras ( )
virtual

Updates the list of cameras on this interface.

This function needs to be called before any cameras can be discovered using GetCameras(). ManagedSystem::GetCameras() will automatically call this function for each interface it enumerates. If the list changed between this call and the last time UpdateCameras was called then the return value will be true, otherwise it is false.

See Also
ManagedSystem::GetCameras()
Returns
true if cameras changed on interface and false otherwise.
